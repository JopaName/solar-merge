# Solar Merge: Империя Энергии

HTML5 merge-idle игра о строительстве солнечной энергетической империи. Разработана на Phaser 3 для платформы Яндекс Игры.

## Особенности

- **Merge механика** — объединение солнечных панелей 1–10 уровня
- **Система заказов городов** — выполняйте заказы и зарабатывайте монеты
- **Магазин панелей** — покупайте панели за монеты
- **Визуализация города** — ваш город растёт с каждым выполненным заказом
- **Интеграция с Yandex Games SDK** — облачные сохранения, реклама
- **Облачные сохранения** — прогресс сохраняется в облаке Яндекса
- **Монетизация через рекламу** — rewarded video и межстраничная реклама

## Технологии

- **Phaser 3** — игровой движок
- **JavaScript (ES6)** — модульная архитектура
- **Vite** — сборщик проекта
- **Yandex Games SDK** — облачные сохранения и монетизация

## Установка и запуск

```bash
npm install
npm run dev
```

Игра будет доступна по адресу `http://localhost:3000`.

## Сборка

```bash
npm run build
```

Готовый билд будет в папке `dist/`.

## Арт и ассеты

### Генерация плейсхолдеров

Если в папке `assets/panels/` нет спрайтов, игра использует Rectangle fallback.
Для генерации PNG-плейсхолдеров выполни:

```bash
npm install sharp
node generate_placeholders.js
```

После этого в `assets/panels/` появятся 10 PNG файлов (64×64).

### Замена на реальный арт

1. Сгенерируй спрайты с помощью AI (Midjourney, Stable Diffusion) — см. `ART_GUIDE.md`
2. Скопируй файлы в `assets/panels/` с именами `panel_1.png` … `panel_10.png`
3. Перезапусти игру — спрайты автоматически подгрузятся

### Структура папок

```
assets/
├── panels/     # Спрайты панелей (64×64 PNG)
├── ui/         # Кнопки, иконки UI (опционально)
├── audio/      # Звуковые эффекты (опционально)
└── fonts/      # Шрифты (опционально)
```
